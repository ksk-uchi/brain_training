============================
CMでよくやってる脳トレを作る
============================

背景
======

だって、楽しそう。。。
５バックとかCMでやってるの見たら、
なんかやりたいなーって。

けどDSって高いし、
他のソフトで魅力的なやつがあるわけでもないし。
だからHTML5とか駆使したら案外簡単そうやし、
作っちゃおっかなって思った訳です。


協力
=======

ksk-uchi(僕)はJavaScriptをちょっとかじっているくらいの人間です。
もしHTMLのUI部分を協力して頂けるとすごくうれしくて泣きます。

ちなみに考えている流れは、
1. start画面。バックの数字(難易度)選択が可能。
2. game画面。適当な一桁の数字同士の四則演算が出題されて、解答できる。
3. 問題数がstart画面で選んだバック数に達したらgame開始。
4. game over画面。間違えたらこの画面。出した問題と答えた数字のペアと何問正解したかとstartに戻るボタンを表示。
5. 出来れば音楽もつけたい。。。
